Welcome to Attribuzer

Attribuzer is an open-source, customizable attribution system that lets developers easily provide credit and project details in a flexible, dynamic way. You can configure your own attribution XML file, and use the provided JavaScript to display credits consistently across your website.

How Attribuzer Works

Attribuzer uses an XML file to store attribution details, including project title, creator information, and license details. The JavaScript file then fetches this data and dynamically displays it on your website.
